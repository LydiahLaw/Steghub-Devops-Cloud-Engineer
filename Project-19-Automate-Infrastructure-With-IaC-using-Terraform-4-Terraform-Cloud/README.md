# Project 19 - Automate Infrastructure With IaC using Terraform 4 (Terraform Cloud)

This project migrates the Terraform infrastructure from Project 18 to Terraform Cloud.
All working code lives in the dedicated repo: https://github.com/LydiahLaw/terraform-cloud

## What this project covers
- Migrating Terraform backend from S3 to Terraform Cloud
- Building custom AMIs with Packer (bastion, nginx, wordpress, tooling)
- Configuring dev/test/prod workspaces in Terraform Cloud
- Post-provisioning configuration with Ansible
